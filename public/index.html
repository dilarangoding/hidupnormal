Password bisa dilihat kembali melalui menu status. Selamat menjalani hidup normal!

Aplikasi ini dibuat dengan tujuan untuk membantu Anda menjauh dari akun media sosial yang mulai mengganggu kegiatan di kehidupan normal Anda. Misal, Anda merasa mulai kecanduan dan rasa kecanduan seperti tidak bisa lepas untuk memeriksa notifikasi membuat Anda susah untuk berkonsentrasi dengan pekerjaan Anda.

Cara kerja aplikasi ini adalah, pertama, Anda menentukan mau sampai kapan akan menjalani hidup normal tanpa gangguan media sosial, lalu aplikasi akan memberikan Anda random password yang susah diingat. Anda akan diminta untuk mengganti password yang sudah diberikan dan logout. Setelah selesai diganti, aplikasi akan memberikan sebuah kode untuk Anda simpan. Dengan kode tersebut Anda bisa mendapatkan password Anda kembali melalui tombol status di bawah.

Aplikasi tidak menyimpan password Anda, melainkan Anda sendirilah yang menyimpannya, karena password sebenarnya ada pada kode yang aplikasi berikan, hanya saja di-enkripsi supaya tidak bisa dibaca.
